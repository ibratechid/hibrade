## Deploy ke GitHub dan Vercel

Untuk menyimpan kode ke GitHub:
1. Buka terminal di folder `c:\xampp\htdocs\hibrade`
2. `git init`
3. `git add .`
4. `git commit -m "Initial secure HIBRADE website setup"`
5. Buat repository baru di GitHub lalu tambahkan remote:
   - `git remote add origin https://github.com/username/repo.git`
   - `git push -u origin main`

Untuk deploy ke Vercel:
- Vercel cocok untuk `index.html`, CSS, dan JavaScript statis.
- File PHP tidak akan dijalankan di Vercel, jadi kami mengecualikannya pada `.vercelignore`.
- Gunakan `vercel.json` untuk memastikan deployment sebagai website statis.

> Jika Anda membutuhkan backend PHP penuh, deploy `*.php` ke host yang mendukung PHP, lalu gunakan Vercel hanya untuk front-end statis.
